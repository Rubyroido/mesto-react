# Проект: Место-реакт

## Обзор:
Сервис Mesto.
Интерактивная страница, на которую можно добавлять фотографии, удалять их и ставить лайки.
Добавлять и редактировать информацию о себе, менять аватар.

## Технологии:
Адаптивная верстка с помощью Flex, Grid.
JS React.
ООП.
API.
